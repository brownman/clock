Clock
=====

a skinnable ruby clock using opengl

Requirements
------------

* Bundler
* Ruby-OpenGL
* Chingu & Gosu

Installation
------------

### Gosu

* http://code.google.com/p/gosu/wiki/GettingStartedOnLinux
* http://code.google.com/p/gosu/wiki/GettingStartedOnOsx
* http://code.google.com/p/gosu/wiki/GettingStartedOnWindows

### Clock

    gem install bundler
    bundle install

Usage
-----

    ruby clock_app.rb
